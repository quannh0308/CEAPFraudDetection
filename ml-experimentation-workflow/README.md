# ML Experimentation Workflow

ML experimentation workflow for fraud detection model optimization in SageMaker Studio.

## Project Structure

```
ml-experimentation-workflow/
├── src/                    # Python modules
├── tests/                  # Test suite
│   ├── unit/              # Unit tests
│   └── property/          # Property-based tests
├── notebooks/             # Jupyter notebooks
├── infrastructure/        # CDK infrastructure stacks
└── requirements.txt       # Python dependencies
```

## Setup

Install dependencies:

```bash
pip install -r requirements.txt
```

## Testing

Run tests:

```bash
pytest tests/
```
